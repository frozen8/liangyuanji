# 良缘纪 💞

> 双人备婚修炼养成微信小程序 —— 以游戏化方式陪伴情侣共赴良缘

基于 Taro 4.2.0 + React 18 + TypeScript + 微信云开发构建，让备婚与日常相处变成一场共同的修仙之旅。

---

## 功能特性

### 双人协作系统
- 邀请码绑定双人关系，数据实时同步
- 双方共同管理灵兽、任务、账单和修炼进度
- 支持解除绑定（需双方同意，数据销毁）

### 灵兽窝（首页）
- 姻缘灵兽随双修境界进化（蛋 → 幼兽 → 成兽 → 神兽）
- 饱食 / 心情 / 灵力 / 好感四维状态管理
- 番茄投喂功能，消耗修炼番茄提升灵兽状态
- 婚礼倒计时、今日待办、收支速览

### 降妖录（任务）
- 备婚任务发布与降服（7 类妖兽）
- 防重复提交保护
- 降服获得修为 + 灵石扣减
- 显示任务发布者和降服者

### 灵石簿（账本）
- 灵石池预算管理（总预算 / 已花 / 结余）
- 账单流水按日期倒序，显示花费人
- 6 类分类筛选 + 统计图表
- 双人对比、7 日趋势

### 修炼阁（番茄钟）
- 圆形倒计时，6 个预设方向 + 自定义
- 15/25/50 分钟预设 + 自定义时长
- 暂停 / 继续 / 终止修炼
- 提前终止修为减半，番茄不变
- 修炼历史记录

### 修为与境界系统
- 降服妖兽和修炼共同累加修为
- 6 阶境界：凡人 → 练气 → 筑基 → 金丹 → 元婴 → 化神
- 灵兽形态随境界自动进化
- 境界突破提示

### 自定义 TabBar
- emoji 图标 + 弹跳切换动画
- 底部安全区适配

## 快速开始

### 环境要求

- Node.js ≥ 18
- 微信开发者工具（稳定版）
- 微信小程序账号 + 云开发环境

### 安装与运行

```bash
# 安装依赖
npm install

# 开发模式（热更新）
npm run dev:weapp

# 生产编译
npm run build:weapp
```

### 配置

1. 在 `src/app.tsx` 中填入你的云环境 ID
2. 在 `project.config.json` 中填入你的 AppID
3. 部署云函数（见部署指南）

## 部署

完整的微信公众平台部署流程请参考 [docs/DEPLOYMENT.md](docs/DEPLOYMENT.md)。

主要步骤：
1. 配置 AppID 和云环境 ID
2. 编译小程序 `npm run build:weapp`
3. 部署 23 个云函数到微信云开发
4. 初始化数据库集合
5. 上传代码到微信开发者工具
6. 提交审核并发布
