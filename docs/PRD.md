# 良缘纪 · 产品需求文档（PRD）

> **产品名**：良缘纪（双人备婚修炼养成记）
> **产品定位**：一款游戏化的双人备婚微信小程序，把"待办管理 + 预算记账 + 番茄钟专注"三大枯燥任务，变成情侣共同养育姻缘灵兽、双修飞升的甜蜜旅程。婚礼当天 = 飞升日。
> **目标平台**：微信小程序（基于 Taro 4.x 跨端框架，可同时编译 H5 预览）
> **目标用户**：备婚情侣（首要服务开发者与未婚妻的结婚计划）

---

## 一、产品概述

### 1.1 核心痛点
备婚是情侣共同经历的一段高压期：待办繁杂（婚纱/酒店/婚庆/请柬/礼金…）、预算紧张需要记账、许多事项需要专注推进。传统 TODO / 记账 / 番茄钟工具各自割裂、枯燥乏味，缺乏"两人共同参与"的陪伴感。

### 1.2 解决方案
将三大实用功能用**双人修炼养成**主线串联：

| 原始功能 | 游戏化呈现 | 行为反馈 |
|---------|-----------|---------|
| TODOlist | **降妖录**：备婚任务 = 妖兽（婚纱妖 / 酒店妖 / 喜宴妖 / 请柬妖 / 礼金妖 / 采购妖） | 完成 = 击败妖兽，掉落修为 + 灵石 + 灵兽心情↑；逾期 = 妖兽攻击灵兽 |
| 记账 | **灵石簿**：预算 = 灵石池，支出 = 消耗灵石 | 节省 = 节余转灵石奖励；超支 = 灵兽心情↓；分类流水 + 双人对比 |
| 番茄钟 | **修炼阁**：专注 = 修炼，方向可选 | 完成一个番茄 = 修为 + 灵石；双人同修 = 双修加成 ×1.5 |

辅以**姻缘灵兽养成**与**双修境界**两条主线，让三大功能的行为共同影响灵兽状态与境界进化。

### 1.3 核心价值
- **陪伴感**：双人共同养育同一只灵兽，共享修为与境界
- **趣味性**：枯燥的备婚事务变成降妖修炼，每一步都有奖励反馈
- **实用性**：TODO + 记账 + 番茄钟三合一，无需在多个工具间切换
- **仪式感**：婚礼当天 = 飞升日，备婚过程化为可回顾的修炼纪事

## 二、核心游戏化机制

### 2.1 姻缘灵兽（养成主线）
两人共同养育一只姻缘灵兽，灵兽随双修境界进化形态：

```
姻缘蛋 → 幼兽 → 成兽 → 神兽
```

灵兽四维状态：

| 状态 | 含义 | 影响行为 |
|------|------|---------|
| 饱食度 | 灵兽饥饿程度 | 完成降妖任务喂养 |
| 心情值 | 灵兽愉悦程度 | 任务逾期/超支扣减；节省预算/双修提升 |
| 灵力值 | 灵兽能量程度 | 番茄钟修炼充能 |
| 好感度 | 双人羁绊程度 | 双人协作任务、双修修炼提升 |

状态过低灵兽会"蔫巴"（视觉表现变灰暗），状态满格触发"灵兽暴击"额外奖励。

### 2.2 双修境界（等级系统）

| 境界 | 所需修为 | 解锁内容 |
|------|---------|---------|
| 凡人 | 0 | 初始灵兽蛋 |
| 练气 | 500 | 灵兽孵化为幼兽 |
| 筑基 | 2000 | 灵兽装饰商店 |
| 金丹 | 5000 | 灵兽进化为成兽 |
| 元婴 | 12000 | 双修场景皮肤 |
| 化神 | 25000 | 灵兽进化为神兽 |
| **飞升** | 婚礼当日 | 飞升典礼 + 结算回顾 |

### 2.3 双人协作机制
- 邀请码绑定双人关系，共享同一灵兽与修为
- 任务可"接单 / 协作"，双人完成享加成
- 番茄钟"双修模式"：两人同时修炼，修为 ×1.5
- 记账双人流水，可对比"谁更省"
- 双人共同成就墙

### 2.4 奖励体系
- **修为**：完成任务（基础 + 难度系数）、修炼（每分钟 10 修为）、节省预算（节余 10%）
- **灵石**：完成任务（难度 ×10）、修炼完成（时长 ×2）、节省预算（节余转灵石）
- **灵兽状态**：任务 → 饱食度↑心情↑；修炼 → 灵力↑；节省 → 心情↑

## 三、核心功能点

### 3.1 首页 - 灵兽窝（home）
- 姻缘灵兽展示（CSS 绘制 + 状态气泡）
- 双修境界进度条 + 距下一境界修为
- 灵兽四维状态环形图（饱食 / 心情 / 灵力 / 好感）
- 婚礼倒计时（飞升倒计时）
- 今日待办概览（前 3 项降妖任务）
- 今日修炼时长 / 今日灵石收支速览
- 快速入口：降妖 / 记账 / 修炼

### 3.2 降妖录 - 任务（quest）
- 备婚任务列表，按妖兽类型分类筛选（全部 / 婚纱 / 酒店 / 喜宴 / 请柬 / 礼金 / 采购）
- 任务卡片：妖兽名（任务标题）、难度星级、紧迫度（攻击力）、截止日期、奖励预览
- 状态：待降服 / 进行中 / 已降服 / 已逾期
- 添加任务（创建妖兽）：标题、分类、难度、截止日、协作模式
- 完成任务弹窗：击败动画 + 奖励掉落
- 双人任务协作标记

### 3.3 灵石簿 - 账本（ledger）
- 顶部预算总览：总预算 / 已花 / 结余（灵石池）
- 当月支出统计 + 分类占比环形图
- 账单流水列表（双人记账，标记记账人头像）
- 分类筛选（婚纱 / 酒店 / 餐饮 / 礼金 / 装饰 / 其他）
- 快速记一笔：金额、分类、备注、日期、记账人
- 节省奖励提示 / 超支警告
- 双人记账排行（本月谁更省）

### 3.4 修炼阁 - 番茄钟（cultivate）
- 番茄钟主界面：圆形倒计时 + 当前修炼方向
- 修炼方向选择：备婚相关（选婚纱 / 写请柬 / 规划行程 / 健身备婚等）或自定义
- 时长可选：25min（标准）/ 15min（短修）/ 50min（长修）
- 双修模式：检测伴侣是否在修炼，同修享 ×1.5 加成
- 修炼完成：修为 + 灵石奖励、灵兽灵力充能动画
- 今日修炼统计：完成番茄数 / 总时长 / 获得修为
- 修炼历史入口

### 3.5 仙府 - 我的（mine）
- 双人头像 + 姻缘指数 + 绑定状态
- 双修境界卡片 + 总修为
- 灵兽图鉴入口（已解锁形态展示）
- 成就墙入口（已解锁成就）
- 修炼统计入口
- 邀请绑定 / 解绑
- 婚礼日设置
- 设置（通知 / 关于 / 反馈）

### 3.6 任务详情（task-detail）
- 妖兽卡片：按任务类型渐变色背景，展示类型 emoji、名称、难度星级、状态、紧迫度
- 紧迫度五档算法：已逾期/危急/紧迫/临近/充裕，各自配色与 emoji
- 任务信息区：名称、类型、难度、截止日期、创建时间、接单人、完成时间
- 降服奖励预览：修为 / 灵石 / 心情三项
- 底部操作栏：放弃（删除任务）/ 降服（完成任务）
- 降服成功弹窗：奖励掉落展示，2.5 秒后自动返回

### 3.7 创建任务（task-create）
- 妖兽名称输入（最多 30 字）
- 妖兽类型横向滚动选择（7 类：婚纱/酒店/喜宴/请柬/礼金/采购/其他）
- 难度星级选择（1-5 星，每档显示攻击力）
- 降服期限日期选择器（默认 +7 天）
- 妖兽备注（最多 200 字）
- 预估奖励实时联动：修为=难度×35、灵石=难度×10、心情=难度×5

### 3.8 账单详情（ledger-detail）
- 金额卡片：按分类色渐变，展示负数金额
- 账单信息：金额、分类、日期、记账人、创建时间
- 分类概览：同分类累计金额 + 占总支出百分比 + 进度条
- 底部操作：删除账单 / 编辑（编辑功能待开发）

### 3.9 账单统计（ledger-stats）
- 时间筛选 Tab（全部 / 按月）
- 灵石池总览：已花 / 预算 / 进度条 / 百分比 / 结余或超支
- 分类支出排行：横向条形图 + 排名 + 百分比
- 双人记账对比：双方金额、占比、VS 标识
- 近 7 日支出趋势：CSS 柱状图
- 预算建议：动态生成（>90% 超支预警 / ≥80% 警告 / ≤30% 充足）

### 3.10 修炼历史（cultivate-history）
- 修炼总览：番茄数 / 总时长 / 总修为 / 双修次数（×1.5 加成提示）
- 按日期分组的历史列表（今天 / 昨天 / MM-DD）
- 会话卡片：方向 emoji、方向名、时长、双修 tag、奖励、时间区间
- 点击展开详情：开始/结束时间、方向、时长、双修加成、完成状态
- 支持下拉刷新

### 3.11 灵兽图鉴（beast-pedia）
- 当前灵兽卡：emoji、名称+阶段、境界+等级、修为进度条
- 形态进化图鉴：4 阶段（蛋→幼兽→成兽→神兽），按境界解锁
- 装饰收藏网格：8 个装饰，筑基境（修为≥2000）解锁
- 点击已解锁阶段展开详情：形态阶段、解锁境界、特性描述

### 3.12 成就墙（achievement）
- 进度卡片：已解锁/总数、进度条、百分比、动态激励文案
- 已解锁成就网格：emoji、名称、解锁日期
- 未解锁成就网格：🔒、名称、描述提示
- 详情弹窗：emoji、名称、状态徽章、描述、解锁时间或解锁提示

### 3.13 邀请绑定（invite）
- 三态切换：未绑定 / 邀请码展示 / 已绑定
- 创建关系：选婚礼日期 + 昵称 → 生成邀请码
- 加入关系：6 位邀请码 + 昵称 → 绑定成功
- 邀请码复制 / 微信分享（useShareAppMessage）
- 已绑定态：双方头像 + 昵称 + 绑定时间 + 距飞升日 + 姻缘指数 + 解除绑定

### 3.14 飞升日设置（wedding-day）
- 倒计时卡片：天数 + 婚礼日期
- 修改飞升日：日期选择器 + 保存
- 飞升预告：飞升典礼 / 备婚历程回顾 / 灵兽最终形态 / 修炼记忆相册
- 修炼里程碑时间轴：5 个节点（姻缘初结→灵兽孵化→练气→筑基→飞升登仙）

### 3.15 设置（settings）
- 通知设置：4 个开关（任务到期/修炼/超支预警/伴侣动态）
- 提醒时间：ActionSheet 6 时段选择
- 数据管理：清除缓存 / 导出备婚数据（JSON）/ 重置所有数据
- 关于：版本号 v1.0.0 / 关于弹窗
- 反馈：意见反馈（待开发）/ 评分 / 加入交流群

## 七、设计原则

### 7.1 视觉风格
**修仙古风 × 浪漫温馨**：以暖色调（姻缘粉 / 灵石青 / 修为金）为主，融合修仙元素（云纹 / 仙气 / 灵气）与婚礼浪漫元素（爱心 / 花瓣 / 喜庆），打造既仙气又甜蜜的视觉氛围。

### 7.2 配色方案
- 主题色：姻缘粉 `#FF6B9D`（灵兽 / 婚姻主线）
- 辅助色：灵石青 `#3DAA9A`（记账 / 灵石）、修为金 `#F7B500`（修炼 / 等级）
- 背景色：页面 `#FFF8F3`（暖白）/ 卡片 `#FFFFFF`
- 文本色：主要 `#2B1F2C` / 次要 `#6B5A6E` / 辅助 `#A89AAC`
- 边框色：`#F0E4EA`
- 语义色：成功 `#00B42A` / 警告 `#FF7D00` / 错误 `#F53F3F`

### 7.3 设计规范
- 8px 网格系统，页面左右安全边距 32rpx
- 卡片圆角 16rpx + 轻阴影
- 按钮全圆角胶囊形（48rpx 圆角）
- 字体层级：大标题 40rpx / 标题 34rpx / 副标题 30rpx / 正文 28rpx
- 渐变背景营造仙气氛围

## 八、开发阶段规划
